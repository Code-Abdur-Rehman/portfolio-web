# Developer Portfolio Prototype

A minimalist dark-mode portfolio landing page for a frontend developer.

## Features
- Dark developer theme with neon accent styling
- Hero section with animated typing effect
- Tech stack badges
- Featured project cards with hover interactions
- About section with human-centered copy
- Contact links for email, LinkedIn, and GitHub
- Responsive layout for mobile and desktop

## Files
- `index.html` — homepage structure
- `styles.css` — dark theme styling and responsive layout
- `script.js` — typing effect and section fade-in animation

## Run locally
1. Open `index.html` in your browser.
2. Or use a static server such as `Live Server` or `npx serve`.

## Customization
- Replace placeholder project summaries with your real projects.
- Update the contact URLs to your actual email, LinkedIn, and GitHub.
- If you want a React/Next.js version, this HTML prototype can be ported to a component-based app easily.
